SPGuidePage
===========
专为新手引导功能打造的开源库
-----------
**支持特性：**

* 支持多种镂空，圆、椭圆、矩形、圆角矩形、虚线边框矩形、以及使用cornerRadis字段可自定义的所有shape。
* 支持纯图片添加，也支持图文，更支持指定区域高亮+图文
* 所有位置可自定义，也可以使用默认值，默认状态下，可根据高亮区域的位置做调整，默认在下。
* 所有控件都支持自定义，也有默认可选样式，方便快捷，又不失可扩展性。
* 新增一页一页显示引导块的功能

**后续新特性会持续更新，敬请期待**
